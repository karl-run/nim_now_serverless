# nim serverless function on zeit's now

A small example on how to deploy a serverless function on [now](https://zeit.co/now) using [nim](nim-lang.org) programming language.

## Develop

1. Install nim
2. Run `nimble build`
3. Run `./now_nim_serverless`

## Deploy

1. Install `now-cli`
2. Run `now` to deploy
